# odcs-openapi-example
This repository takes the familiar "Petstore" OpenAPI example and captures the schemas in an [Open Data Contract Standard](https://bitol-io.github.io/open-data-contract-standard/v3.1.0/) specification that is used to generate & publish versioned:

- OpenAPI Spec
- JSON Schemas used instead of in-lining schemas in the OpenAPI Spec & Useful as a general format for codgen in a variety of popular FOSS tools.
- Typescript Classes
- Protobuf files for a hypothetical message payload
